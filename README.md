# Pokemon Nexus

Aplicación web para gestionar entrenadores, explorar Pokémon y registrar capturas.

---

## Funcionalidades

- Registro y gestión de usuarios
- Pokedex con búsqueda, filtros y paginación
- Sistema de capturas de Pokémon
- Visualización de capturas por usuario

---

## Tecnologías usadas

- Node.js
- Express
- Sequelize
- MySQL
- HTML, CSS, JavaScript

---

## Cómo ejecutar el proyecto

1. Instalar dependencias:
```bash
npm install
```

2. Crear archivo `.env` con los datos de la base de datos:
```
DB_NAME=
DB_USER=
DB_PASSWORD=
DB_HOST=
DB_DIALECT=mysql
```

3. Ejecutar el servidor:
```bash
npm start
```
